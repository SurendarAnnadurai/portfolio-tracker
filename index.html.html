or click to browse — .csv and .txt supported

Your CSV must use these exact column names in the header row. Column order does not matter.

market,ticker,name,type,date,qty,price,currency,fx_to_eur,notes
market  →  India US Europe Crypto
type  →  buy sell dividend bonus  — case insensitive (Buy, BUY, buy all work)
date  →  YYYY-MM-DD  e.g. 2024-03-15
price  →  price per share in original currency. For dividend: total dividend received. For bonus: use 0
fx_to_eur  →  leave blank to auto-fetch the real historical rate for that exact date. Or enter manually: EUR=1, USD≈0.92, INR≈0.011, GBP≈1.16
qty  →  number of shares. Can be left blank for dividend rows
⚠ If your numbers contain commas (e.g. 1,470) — remove them. Use 1470 instead. Or wrap the value in quotes: "1,470"
